## “buddy” Package

This package is to guess and test Buddy’s nicknames. Buddy is my
magnificent cat.

## Installation

You can install the released version of buddy with:

``` r
install.packages("remotes")
remotes::install_github("asmae-toumi/buddy")
```

## Example

``` r
library(buddy)
is_buddy("bud")
```
